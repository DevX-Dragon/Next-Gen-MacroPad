# NextGen Macropad
The Next-Gen MacroPad is a fully DIY, multi-purpose macro pad kit. It's got programmable keys, it looks cool, and most importantly — I made it. A 14-year-old made it. So if it breaks, don't be too surprised, but also it probably won't break because I tested it a bunch.

## Specifications
- 16 Programmable Keys
- 0.96 OLED Display
- Both Bluetooth and USB C connectivity
- Per Key RGB

## Whats inside?
- `/3D Files` : 3D Files for 3D Printing and Source Files for your own modifcations.
- `/Firmware` : Firmware for the Macro Pad (CircuitPyhton)
- `/KiCad Files` : Soruce files for KiCad for your own modifications.
- `/Images` : Previews of multiple assets
- `/Production` : Prodution files for PCB manufacturings.
- `BOM.csv` : Price Breakdown and Links
